Appendix C. Change history

Changes from version 1.2

This version of the specification contains the changes listed below compared to the previous specification version. Typos, grammatical and formatting changes are not listed.

Appendix A: Deploying JSR 180 on MIDP 2.0 Platform

  • CDC Security: Included permission class definitions for permission class based implementations, such as those based on CDC. Earlier permission definitions had errors and were incorrectly included in the specification javadoc although they are not required in implementations that use permission names.

  • JSR 180 Push Registration Entry: Added the missing transport parameter and clarified the default to be UDP for SIP connections.

Appendix A: Security
